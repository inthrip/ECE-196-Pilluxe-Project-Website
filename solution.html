Our Proposed Solution

This page presents the proposed design, the main technical approach, and the overall system structure.

Solution Overview

Our solution is a smart pillbox with four pill containers, each equipped with LEDs and a button. A PCB underneath connects the system to Wi-Fi and Bluetooth, allowing a mobile app to manage medication schedules, send reminders, light the correct container, and record when pills are taken. The app also uses features like daily streaks to encourage consistency.

Generated Concept Sketch

What the Technology Will Do

  • Store four different pill types in separate rectangular containers.
  • Use LEDs to indicate which pill container should be accessed at the scheduled time.
  • Send schedule-based reminders through the connected mobile app.
  • Detect button presses to confirm the correct medication was taken.
  • Track medication-taking behavior and reward consistency with features such as daily streaks.

Technological Aspects of the Solution

Hardware

  • Four rectangular pill containers
  • Five LEDs and one button on each container face
  • Physical enclosure for organized medication storage

PCB / Connectivity

  • PCB mounted underneath the containers
  • Wi-Fi connectivity for app communication
  • Bluetooth connectivity for direct device interaction

Software / App Logic

  • User-configured medication schedules
  • Phone notifications when medication is due
  • Button-press confirmation and streak-based incentives
